Features & how to use

Drop this file into the root of your repository, enable GitHub Pages in the repository settings, and pick the branch to publish. No build steps required.

Theming

Toggle between dark and light modes. Colors are controlled by CSS variables at the top of the file for easy customization.

Customization

Replace text, tweak the gradient variables, or add your own sections. Keep everything in this file for portability.
